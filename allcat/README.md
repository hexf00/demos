## 如何使用

方法1：npm启动

1. `git clone https://github.com/hexf00/demos`
2. `cd demos/allcat`
3. `npm install`
4. 访问 http://127.0.0.1:8000

说明：系统需安装 node.js ，本版本仅供体验，数据存储在浏览器端，使用过程请记得导出备份
